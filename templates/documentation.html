Getting Started with DeepMed

Welcome to DeepMed, the privacy-first, no-code AI platform built for medical professionals. This guide will help you understand how to use our platform effectively.

Creating Your Account

To begin using DeepMed, you'll need to create an account:

  1. Click on "Get Started" in the navigation bar
  2. Enter your name, email address, and password
  3. Click "Create Account"
  4. Log in with your new credentials

Dashboard Overview

Once logged in, you'll be presented with your dashboard, which provides access to all DeepMed's capabilities:

  • Tabular: For working with tabular data (spreadsheets)
  • Images: For working with medical imaging data
  • Regression: For predicting numerical values

Each section contains tools for training models, viewing your trained models, and making predictions.

Tabular Data Models

Tabular models are ideal for structured data like patient records, lab results, and clinical measurements.

Preparing Your Data

For best results, prepare your tabular data with the following guidelines:

  • Use CSV or Excel format (.csv, .xlsx, .xls)
  • Include a header row with column names
  • Ensure your target variable (what you want to predict) is clearly identified
  • Remove any personally identifiable information (PII) for privacy

Training a Classification Model

To train a classification model:

  1. Navigate to "Tabular → Training Classification" in the main menu
  2. Upload your dataset using the file upload button
  3. Select the target column (what you want to predict)
  4. Choose the columns to include in your model (or use automatic selection)
  5. Click "Train Model" to begin the training process

DeepMed will automatically:

  • Clean your data by handling missing values and outliers
  • Select the most relevant features if you opted for automatic selection
  • Train multiple model types simultaneously (Random Forest, XGBoost, etc.)
  • Evaluate and compare model performance
  • Select the best-performing model for deployment

Image Models

DeepMed's image models are designed for medical imaging classification, such as identifying conditions from X-rays, MRIs, CT scans, and other medical imagery.

Preparing Your Images

For optimal performance:

  • Organize images into folders, with each folder representing a class/category
  • Use common image formats (.jpg, .png, .dicom)
  • Ensure images are of similar dimensions and quality
  • Remove any patient identifiers from images

Image Augmentation

If you have limited training data, our augmentation feature can help:

  1. Navigate to "Images → Augment" in the main menu
  2. Upload your original images
  3. Select augmentation techniques (rotation, flip, contrast adjustment, etc.)
  4. Specify how many variations to generate
  5. Click "Generate Augmentations"

This creates additional training examples by applying realistic modifications to your images.

Training an Image Classification Model

To train an image classification model:

  1. Navigate to "Images → Pipeline" in the main menu
  2. Upload your image dataset (organized in class folders)
  3. Configure preprocessing options
  4. Select model architecture (default is fine for most use cases)
  5. Click "Train Model"

Regression Models

Regression models predict numerical values instead of categories, making them ideal for forecasting lab values, estimating patient outcomes, or predicting treatment effectiveness scores.

Training a Regression Model

To train a regression model:

  1. Navigate to "Regression → Train Regression" in the main menu
  2. Upload your dataset (in CSV or Excel format)
  3. Select the numerical target column you want to predict
  4. Choose which features to include (or use automatic selection)
  5. Click "Train Model"

The process includes automatic data cleaning, feature selection, and model benchmarking to find the most accurate regression model for your data.

Anomaly Detection

Anomaly detection identifies unusual patterns in data that don't conform to expected behavior, which is particularly useful for identifying rare conditions or abnormalities in medical images.

Using Anomaly Detection

To utilize anomaly detection:

  1. Navigate to "Images → Anomaly Detection" in the main menu
  2. Upload a set of "normal" images for the model to learn from
  3. Configure detection sensitivity
  4. Train the anomaly detection model
  5. Upload test images to identify potential anomalies

The system will highlight regions that deviate significantly from the learned "normal" patterns.

Data Privacy

DeepMed is built with privacy as a core principle, ensuring your medical data remains secure and compliant.

Privacy Measures

  • Local Processing: All data processing happens locally, with no external data sharing
  • End-to-End Encryption: Data is encrypted during transit and at rest
  • No Data Retention: We don't store your medical datasets beyond processing
  • Compliance: Our system is designed to meet GDPR and HIPAA requirements

Best Practices

To maximize privacy protection:

  • Remove direct patient identifiers before uploading
  • Use anonymized or de-identified datasets when possible
  • Regularly review and delete unused models and predictions
  • Ensure you have proper authorization to use the data for AI analysis

Frequently Asked Questions

General Questions

What makes DeepMed different from other AI platforms?

DeepMed is specifically designed for healthcare, with privacy-first architecture, no-code interfaces, and automated workflows tailored to medical data. Unlike general-purpose AI tools, we optimize for healthcare use cases and compliance requirements.

Do I need coding experience to use DeepMed?

No. DeepMed is designed to be fully accessible to non-technical users with no coding required. Our intuitive interface guides you through the entire process from data upload to prediction.

How accurate are the models?

Model accuracy depends on your data quality and the complexity of the prediction task. DeepMed automatically benchmarks multiple algorithms and selects the best performer for your specific dataset. We typically achieve competitive or superior performance compared to manually developed models.

How do I get help if I encounter issues?

Contact our support team at support@deepmed.ai for assistance. We typically respond within 24 hours on business days. For urgent issues, you can use the live chat function available in the platform.
