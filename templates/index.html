Getting Started
  1. Prepare Your Dataset
    Organize your medical images in a ZIP file with folders for each category.
  2. Upload to Image Analysis
    Navigate to the Images page and upload your dataset ZIP file.
  3. Train Your Model
    Choose an analysis type and privacy settings, then start training.
  4. Download and Deploy
    Download your trained model for use in your own applications.
Quick Tips
Privacy Notice: Enable Privacy Mode to process images locally and only send extracted features to the server.
Dataset Structure

For best results, use a ZIP file with the following structure:

dataset.zip
├── class1/
│   ├── image1.jpg
│   ├── image2.jpg
│   └── ...
├── class2/
│   ├── image1.jpg
│   ├── image2.jpg
│   └── ...
└── ...
Model Selection
