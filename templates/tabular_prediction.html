Tabular Prediction with Custom Models

Use Custom Model Packages

Upload a model package (ZIP file) containing your trained model and required dependencies, along with tabular data to generate predictions.

The model package should include:

  • predict.py script that processes the input
  • requirements.txt listing the required dependencies
  • Any model weights or additional files needed

The system will output the prediction results as a CSV file for download.

Target Column Selection Feature

When uploading your data file, you can now select a specific target column for predictions:

  • What it does: Tells the model which column contains the values you want to predict
  • When to use: Use this feature when working with categorical data where the model needs to know which column contains the class labels
  • Auto-detect: If you don't select a column, the system will try to determine the correct column automatically based on the model package
