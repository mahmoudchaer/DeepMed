Deployment Instructions

Follow these steps to deploy your selected model:

  1. Choose one of the top models above based on your preferred metric:
    • Accuracy: Best for overall performance when all classes are equally important
    • Precision: Best when minimizing false positives is important
    • Recall: Best when minimizing false negatives is critical
  2. Click the "Deploy as API" button to create a containerized API for the model
  3. Wait for the deployment to complete (usually takes 15-30 seconds)
  4. You'll be redirected to a page with instructions on how to use your deployed model API
Note: Each deployed model runs in its own Docker container. Make sure Docker is running on your system before deploying.
